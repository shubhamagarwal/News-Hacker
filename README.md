## Prerequisites

```
node (version > 10)
npm

```

## Heroku link

https://news-hack-clone.herokuapp.com/

## Tech Stack

```
React JS
Redux
NodeJS
SSR
Style Components
```

# Run development project locally

```
Clone https://github.com/shubhamagarwal/News-Hacker.git
npm install
npm run dev
open http://localhost:3000 to view project on the browser
```


# Run Test Cases

```
npm run test
```
